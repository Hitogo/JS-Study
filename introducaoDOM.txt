DOM -> Document Object Model
DOM eh um modelo de objetos para documentos, conjunto de objetos dentro do navegador que dah acesso aos componentes internos do website

Arvore DOM -> janela do navegador
Dentro de window -> location(url) , document(html) e history(de onde veio p onde vai)
